Bootstrap:Image Responsive Class

Apr 25th, 2014

Images sizes are critical in making a responsive website. Fixing the width and height of your image won’t be able to make it respond to the screen size once the screen is resized. So, Bootstrap has an answer to that problem. It is the ‘img-responsive’ class that you can add to the class of images that you will use to your website. This will set the width of the image to 100% and set the height to auto.
